Pour installer WordPress, j'ai commencé par télécharger le fichier zip de WordPress. Ensuite, j'ai extrait le contenu du fichier zip dans le dossier que j'ai préalablement cloné depuis la repository, nommé "Cafeine", et qui se trouve dans le répertoire htdocs de XAMPP. Pour pouvoir utiliser WordPress, j'ai créé une base de données nommée "cafeine" et j'ai établi une connexion entre cette base de données et le dossier "Cafeine" en suivant les étapes fournies par WordPress. Pour faciliter la création de pages, j'ai installé l'extension Elementor, qui permet de concevoir des pages en utilisant des outils de glisser-déposer. J'ai également installé l'extension Woocommerce, qui permet de gérer les produits et les catégories. Enfin, j'ai installé le thème envo-ecommerce pour personnaliser l'apparence de mon site WordPress.
